# 📚 OCR Book Platform

Професионална веб-платформа за оптичко препознавање на текст од слики: качување на документ или страница како слика, автоматско OCR-извлекување на текст, уредување во прегледувач и извезување како PDF.

---

## ✨ Преглед

Платформата ги поврзува **backend API** за обработка на слики и текст со **современ React интерфејс**, со цел едноставно работно искуство од качување до финален документ.

---

## 🛠 Технологии

- **FastAPI** — високоперформансен Python REST API  
- **React** — кориснички интерфејс (Vite + Tailwind CSS)  
- **Tesseract OCR** — препознавање на текст од слики  
- **ReportLab** — генерирање на PDF од уреден текст  

---

## 📋 Предуслови

Пред да го стартувате проектот, потребни се инсталации на:

- **Python** 3.10 или понова верзија (`pip`)
- **Node.js** и **npm** (за frontend)
- **Homebrew** на macOS (за инсталација на Tesseract преку `brew`)

---

## 🔤 Инсталација на Tesseract на macOS

На компјутери со macOS и инсталиран **Homebrew**, инсталирајте го Tesseract OCR со следнава команда во терминал:

```bash
brew install tesseract
```

По инсталацијата, проверете дали е достапен на системскиот `PATH` (на пример со `tesseract --version`). Backend-от користи **pytesseract**, кој очекува да го пронајде извршниот фајл на Tesseract на вашиот систем.

---

## ⚙️ Стартување на FastAPI backend

Од **коренот на проектот** извршете ги чекорите подолу.

1. **Виртуелно окружување** (препорачано):

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. **Зависности:**

   ```bash
   pip install -r requirements.txt
   ```

3. **Покренување на серверот:**

   ```bash
   uvicorn backend.main:app --reload
   ```

API по подразбирање е достапен на **http://127.0.0.1:8000**. Интерактивна документација (Swagger UI): **http://127.0.0.1:8000/docs**.

---

## 💻 Стартување на React frontend

Од терминал, поминете во папката **`frontend`**, инсталирајте ги пакетите и стартувајте го серверот за развој:

```bash
cd frontend
npm install
npm run dev
```

Приложението во режим за развој обично се отвора на **http://localhost:5173**. За OCR и PDF функционалности, **backend мора паралелно да работи**.

Опционално: во `frontend` може да се додаде `.env` со **`VITE_API_URL=http://127.0.0.1:8000`** ако API адресата се менува.

---

## 🚀 Работен тек на кратко

1. Инсталирајте **Tesseract** (`brew install tesseract` на Mac).  
2. Покренете **FastAPI** со `uvicorn backend.main:app --reload`.  
3. Покренете **React** со `npm run dev` во папката `frontend`.  
4. Отворете го интерфејсот во прегледувач, качете слика и продолжете со OCR и извезување PDF.
